# Iot-enabled-Walking-stick-for-visually-impaired
Iot enabled Walking stick for visually impaired

Iot enabled walking stick for the blind has been set up to sense the obstacles around the person. 

Components used:
Ultrasonic sensor
Buzzer
Bread Board
Esp8266(Wi-Fi module) and adapter
Connecting Wires
Arduino UNO
